Brainfrak
=========

Brainfrak is a [Brainfuck](http://en.wikipedia.org/wiki/Brainfuck) interpreter. Brainfuck is an esoteric programming language created by Urban Müller in 1993.

Building Brainfrak
------------------

This project uses [Premake](http://industriousone.com/premake) version 4.4 or higher to generate build scripts for most IDE. Please learn about Premake on their [documentation page](http://industriousone.com/what-premake).

Generated project files are located inside the `project` folder.

Learning Brainfuck
------------------

The [Wikipedia page](http://en.wikipedia.org/wiki/Brainfuck) on the subject is quite extensive, and should teach you what is to know about Brainfuck.
